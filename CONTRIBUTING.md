# Contributing Guidelines

Thank you for your interest in contributing to the AI chatbot project. We appreciate your help in making this project a success.

## Code of Conduct
* Be respectful and considerate of others
* Avoid using offensive language or tone
* Keep discussions focused on the topic at hand

## Contribution Steps
1. Fork the repository and create a new branch for your feature or bug fix
2. Make your changes and commit them with a clear and descriptive commit message
3. Open a pull request and wait for review

## Reporting Issues
If you encounter any issues or bugs, please report them in the issues section of this repository. Be sure to include as much detail as possible to help us diagnose the problem.
